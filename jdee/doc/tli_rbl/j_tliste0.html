C'est la version "RBL" de tListe.
Elle permet lit ses donnees dans un fichier et permet d'afficher une image en premiere position.
C'est la version principale de tListe (la plus complete), dans un desir de compatibilite avec les versions ASP et PERL, le format du fichier est legerement modifie pour gerer un separateur.
Cette version peut egalement lire directement ces donnes dans les parametres de l'applet (sans fichier).
